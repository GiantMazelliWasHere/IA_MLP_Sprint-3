ETAPAS DO
PROJETO

01
CARREGAMENTO E EXPLORACAO

Leitura do CSV, analise de shape (374 x 13), verificacao de nulos e distribuicao das classes target.

02
PRE-PROCESSAMENTO

Extracao de BP_Systolic e BP_Diastolic da coluna "Blood Pressure". Encode de categoricas (Gender, Occupation, BMI) via LabelEncoder. Target fillna("None").

03
DIVISAO TREINO / TESTE

80% treino / 20% teste com estratificacao por classe para preservar a distribuicao original. random_state=42 para reproducibilidade.

04
NORMALIZACAO

StandardScaler ajustado apenas no treino (fit_transform) e aplicado no teste (transform) para evitar data leakage.

05
TREINAMENTO MLP

Rede com camadas (64, 32), ativacao ReLU, otimizador Adam, regularizacao L2 (alpha=0.001) e early stopping com validacao de 10%.

06
AVALIACAO

Acuracia no conjunto de teste + Relatorio de classificacao (precision, recall, F1 por classe) + Cross-validation 5-fold para validacao robusta.

07
IMPORTANCIA DAS FEATURES

Permutation Importance para identificar quais habitos mais impactam o risco — guia as recomendacoes no app.

# Estrutura do modelo MLP
from sklearn.neural_network import MLPClassifier
from sklearn.preprocessing  import StandardScaler

# Normalizacao das features
scaler   = StandardScaler()
X_train  = scaler.fit_transform(X_train)
X_test   = scaler.transform(X_test)

# Definicao da MLP
# Entrada (11) -> Oculta (64, ReLU) -> Oculta (32, ReLU) -> Saida (3)
mlp = MLPClassifier(
    hidden_layer_sizes = (64, 32),
    activation         = "relu",
    solver             = "adam",
    alpha              = 0.001,       # L2 regularizacao
    learning_rate_init = 0.001,
    max_iter           = 300,
    early_stopping     = True,
    validation_fraction= 0.1,
    n_iter_no_change   = 20,
    random_state       = 42
)

mlp.fit(X_train, y_train)

# Avaliacao
y_pred = mlp.predict(X_test)
acc    = accuracy_score(y_test, y_pred)
print(f"Acuracia: {acc:.4f}")
print(classification_report(y_test, y_pred,
      target_names=le_target.classes_))
